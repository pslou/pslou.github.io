If $a^n -1$ is a prime, then $a = 2$ and $n$ is a prime.
Recall that $x^n - y^n= (x-y)(x^{n-1} + x^{n-2}y + \ldots + y^{n-1})$. Then $$a^n -1 = (a-1)(a^{n-1} + a^{n-2} + \ldots + 1).$$ Since $a^n - 1$ is prime, either $a - 1 = a^n -1$ or $a-1 = 1$. If $a -1 = a^n -1$, then $a^n - a = 0$ which implies that $a^{n-1} = 1$ so $a = 1$. Then it must be that $a - 1 = 1$ so $a = 2$.

Now suppose $2^n - 1$ is prime. Let $k, l$ be any factoring of $n$ so that $n = kl$ where $k, l$ are positive integers $k, l$. Then $2^n - 1 = (2^{k})^l - 1^l$ and $$2^n - 1 = (2^k - 1)(2^{k(l-1)} + 2^{k(l-2)} + \ldots + 1).$$ Since $2^n - 1$ is prime, either $2^k -1 = 2^n -1 $ or $2^k - 1 = 1$. if $2^k -1 = 2^n - 1$ then $k = n$ implying $l = 1$. If $2^k - 1 = 1$ then $k = 1$ so $l = n$. Then the only factoring of $n$ is $1 \cdot n$ therefore $n$ is prime.
These primes of the form $2^n - 1$ for positive integer $n$ have name: Mersenne primes. Mersenne primes are named after a French monk Marin Mersenne who lived from 1588-1648. An open question is whether there are infinitely many Mersenne primes.
